- Servlet Filter
  - HTTP request -> WAS -> filter -> servlet(DispatcherServlet)-> controller
  - filter chain
    - HTPP request -> WAS -> filter1 -> filter2 -> servlet -> controller
  - Servlet Container registered filter(singleton instance)

```java
public interface Filter {
    
    //when servlet container init, call this method
    public default void init(FilterConfig filterConfig) throws ServletException {}

    // per request
    public void doFilter(ServletRequest request, ServletResponse response,
            FilterChain chain) throws IOException, ServletException;

    // when servlet container destroy, call this method
    public default void destroy() {}
}
```
